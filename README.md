## Installation
After cloning this repository, migrate to ```tiktok_clone_ui``` folder. Then, follow the following steps:
- Create and Run Android or iOS Emulator/Simulator

Then run the following commands to run your app:
```bash
  flutter pub get
  flutter run
```
